# 8th floor

8th Floor is a modern streetwear brand founded by a group of passionate young creators from Malaysia who aspire to redefine contemporary fashion through originality and bold design. The brand’s website, built using React, Vite, and TypeScript, reflects this vision by combining cutting-edge web technology with a clean, responsive, and dynamic user experience. Leveraging React’s component-based architecture and Vite’s fast development environment, the site delivers smooth navigation, quick load times, and an engaging interface that showcases 8th Floor’s unique collections. TypeScript ensures the project’s reliability and scalability, allowing for maintainable, high-quality code as the brand continues to grow. The result is a digital space that captures the essence of 8th Floor—fresh, youthful, and unapologetically original.
